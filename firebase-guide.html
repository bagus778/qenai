Step 1 of 7

Buat Project Firebase Baru

Buka link ini di tab baru lalu ikuti instruksi berikut.

Buka Firebase Console
  1. Klik tombol Add Project atau Tambahkan Project
  2. Isi nama project dengan: sanz-store
  3. Matikan Google Analytics karena tidak diperlukan
  4. Klik Create Project dan tunggu hingga selesai
  5. Setelah selesai klik Continue

Aktifkan Firestore Database

Sekarang aktifkan database Firestore untuk menyimpan data produk dan order.

Buka Firebase Console
  1. Di sidebar kiri klik menu Build
  2. Klik Firestore Database
  3. Klik tombol Create Database
  4. Pilih Start in Test Mode agar tidak ada error permission
  5. Pilih region asia-southeast1 yang paling dekat dengan Indonesia
  6. Klik Enable dan tunggu hingga database selesai dibuat

Aktifkan Firebase Storage

Aktifkan Storage untuk menyimpan foto dan video tutorial produk.

Buka Firebase Console
  1. Di sidebar kiri klik menu Build
  2. Klik Storage
  3. Klik tombol Get Started
  4. Pilih Start in Test Mode
  5. Pilih region asia-southeast1
  6. Klik Done dan tunggu hingga selesai

Daftarkan Web App dan Salin Config

Daftarkan aplikasi web untuk mendapatkan konfigurasi Firebase.

Buka Firebase Console
  1. Di halaman utama project klik ikon web yang berbentuk tanda lebih dari dan kurang dari </>
  2. Isi App nickname dengan: sanz-store-web
  3. Jangan centang Firebase Hosting
  4. Klik Register App
  5. Salin seluruh isi firebaseConfig yang muncul
  6. Tempel di kotak input di bawah ini

Set Firestore Security Rules

Atur rules Firestore agar website bisa membaca dan menulis data.

Buka Firebase Console
  1. Di Firebase Console buka Firestore Database
  2. Klik tab Rules di bagian atas
  3. Hapus semua rules yang ada
  4. Salin rules di bawah ini dan tempel di editor rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
  1. Klik Publish dan tunggu hingga tersimpan

Set Storage Security Rules

Atur rules Storage agar website bisa upload dan mengambil file.

Buka Firebase Console
  1. Di Firebase Console buka Storage
  2. Klik tab Rules di bagian atas
  3. Hapus semua rules yang ada
  4. Salin rules di bawah ini dan tempel di editor rules
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if true;
    }
  }
}
  1. Klik Publish dan tunggu hingga tersimpan
